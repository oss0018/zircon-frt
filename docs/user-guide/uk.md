# Zircon FRT — Посібник користувача

## Огляд

Zircon FRT (Forensic Research Tool) — платформа для збору та аналізу даних OSINT. Дозволяє аналітикам завантажувати документи, виконувати повнотекстовий пошук по їх вмісту, відстежувати бренди на предмет тайпсквоттингу та моніторити підозрілі індикатори.

---

## Початок роботи

### Вхід до системи

1. Перейдіть за URL вашого екземпляра Zircon FRT
2. Введіть **email** та **пароль**
3. Натисніть **Увійти**

Для створення нового облікового запису натисніть **Створити акаунт** на сторінці входу.

---

## Панель управління

Панель управління надає зведену інформацію:

| Метрика | Опис |
|---------|------|
| Всього файлів | Усі завантажені файли в системі |
| Проіндексовано | Файли, доступні для пошуку |
| Сповіщень сьогодні | Сповіщення за останні 24 години |
| Активних моніторів | Активні елементи списку спостереження |

---

## Завантаження файлів

1. Перейдіть до розділу **Файли** у бічному меню
2. Натисніть **Завантажити файл** або перетягніть файл у зону завантаження
3. Підтримувані формати: `.txt`, `.log`, `.md`, `.csv`, `.json`, `.sql`, `.pdf`, `.docx`, `.doc`, `.xlsx`, `.xls`, `.xml`
4. Після завантаження файл ставиться в чергу індексування
5. Після індексування файл стає доступним для пошуку

### Управління файлами

- **Перегляд**: натисніть на рядок файлу для перегляду метаданих і попереднього перегляду тексту
- **Завантажити**: натисніть значок завантаження для отримання оригінального файлу
- **Видалити**: натисніть значок кошика для видалення файлу

---

## Пошук файлів

1. Перейдіть до розділу **Пошук** у бічному меню
2. Введіть запит у рядок пошуку
3. Використовуйте **фільтри** зліва для звуження пошуку за типом файлу або діапазоном дат
4. Результати відсортовані за релевантністю з підсвічуванням збігів

### Синтаксис запитів

| Синтаксис | Приклад | Опис |
|-----------|---------|------|
| Звичайний текст | `звіт шкідливе ПЗ` | Збіг будь-якого слова |
| Точна фраза | `"командний сервер"` | Точний збіг фрази |
| AND | `фішинг AND облікові дані` | Обидва терміни мають бути присутні |
| OR | `шифрувальник OR вайпер` | Будь-який з термінів |
| NOT | `звіт NOT резюме` | Виключити термін |
| Підстановочний знак | `pass*` | Пошук за префіксом |

### Експорт результатів пошуку

Після виконання пошуку натисніть кнопки експорту над результатами:

- **Експорт CSV** — файл з роздільниками-комами
- **Експорт JSON** — структурований JSON з усіма метаданими
- **Експорт PDF** — форматований PDF-звіт

## Захист бренду

Модуль захисту бренду відстежує тайпсквоттинг і фішингові домени, що імітують ваш бренд.

### Додавання спостереження за брендом

1. Перейдіть до розділу **Захист бренду**
2. Натисніть **Додати бренд**
3. Заповніть:
   - **Назва бренду**: відображувана назва (наприклад, «ТОВ Акме»)
   - **Офіційний URL**: ваш домен (наприклад, `https://acme.ua`)
   - **Ключові слова**: через кому
   - **Поріг схожості**: мінімальний % схожості для генерації сповіщення (за замовчуванням: 70%)
   - **Розклад сканування**: Щогодини, Щодня або Щотижня
4. Натисніть **Додати бренд**

### Перегляд сповіщень

1. Натисніть **Деталі** на картці бренду
2. У таблиці сповіщень показано:
   - **Знайдений домен**: виявлений підозрілий домен
   - **Схожість**: наскільки він схожий на ваш бренд (круговий індикатор)
   - **Джерела**: DNS, WHOIS, SSL, пошукові системи
   - **Статус**: Нове / Перевірено / Відхилено
3. Дії:
   - **Перевірено**: позначити як досліджене
   - **Відхилити**: ігнорувати хибне спрацювання
   - **Відкрити знову**: повернути відхилене сповіщення

### Ручне сканування

Натисніть **Глибоке сканування** у детальному перегляді для негайного запуску сканування.

### Експорт сповіщень

У детальному перегляді бренду використовуйте кнопки експорту (CSV, JSON, PDF).

## Моніторинг (Список спостереження)

Відстежуйте конкретні індикатори: IP-адреси, домени, email-адреси, хеші.

### Додавання елемента

1. Перейдіть до розділу **Моніторинг**
2. Натисніть **Додати елемент**
3. Виберіть **тип**: IP, Домен, Email, Хеш, Ключове слово
4. Введіть **значення**
5. Виберіть **інтервал перевірки**
6. Натисніть **Зберегти**

---

## Сповіщення

Сповіщення в реальному часі відображаються у значку дзвіночка (вгорі праворуч).

- Натисніть дзвіночок для перегляду всіх сповіщень
- Натисніть **Позначити всі як прочитані** для скидання лічильника

---

## Налаштування

### Профіль

- Оновіть відображуване ім'я та email
- Змініть пароль

### Інтеграції

Підключіть зовнішні сервіси: Email (SMTP), Webhook, Telegram.

## Усунення несправностей

**Пошук не повертає результатів**
- Переконайтеся, що файл проіндексовано (розділ «Файли» — зелений індикатор)
- Спробуйте спростити запит

**Сканування бренду не показує сповіщень**
- Спробуйте знизити поріг схожості до 60%
- Запустіть **Глибоке сканування** вручну

**Завантаження файлу завершується помилкою**
- Перевірте максимальний розмір файлу (за замовчуванням: 100 МБ)
- Переконайтеся, що формат файлу підтримується
