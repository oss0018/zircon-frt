# Zircon FRT — Руководство пользователя

## Обзор

Zircon FRT (Forensic Research Tool) — платформа для сбора и анализа данных OSINT. Позволяет аналитикам загружать документы, выполнять полнотекстовый поиск по их содержимому, отслеживать бренды на предмет тайпсквоттинга и мониторить подозрительные индикаторы.

---

## Начало работы

### Вход в систему

1. Перейдите по URL вашего экземпляра Zircon FRT
2. Введите **email** и **пароль**
3. Нажмите **Войти**

Для создания нового аккаунта нажмите **Создать аккаунт** на странице входа.

---

## Панель управления

Панель управления предоставляет сводную информацию:

| Метрика | Описание |
|---------|----------|
| Всего файлов | Все загруженные файлы в системе |
| Проиндексировано | Файлы, доступные для поиска |
| Алерты за сегодня | Алерты за последние 24 часа |
| Активных мониторов | Активные элементы списка наблюдения |

---

## Загрузка файлов

1. Перейдите в раздел **Файлы** в боковом меню
2. Нажмите **Загрузить файл** или перетащите файл в зону загрузки
3. Поддерживаемые форматы: `.txt`, `.log`, `.md`, `.csv`, `.json`, `.sql`, `.pdf`, `.docx`, `.doc`, `.xlsx`, `.xls`, `.xml`
4. После загрузки файл ставится в очередь индексирования
5. После индексирования файл становится доступным для поиска

### Управление файлами

- **Просмотр**: нажмите на строку файла для просмотра метаданных и предпросмотра текста
- **Скачать**: нажмите значок загрузки для получения оригинального файла
- **Удалить**: нажмите значок корзины для удаления файла

---

## Поиск файлов

1. Перейдите в раздел **Поиск** в боковом меню
2. Введите запрос в строку поиска
3. Используйте **фильтры** слева для сужения поиска по типу файла или диапазону дат
4. Результаты отсортированы по релевантности с подсветкой совпадений

### Синтаксис запросов

| Синтаксис | Пример | Описание |
|-----------|--------|----------|
| Обычный текст | `отчёт по вредоносам` | Совпадение любого слова |
| Точная фраза | `"командный сервер"` | Точное совпадение фразы |
| AND | `фишинг AND учётные данные` | Оба термина должны присутствовать |
| OR | `шифровальщик OR вайпер` | Любой из терминов |
| NOT | `отчёт NOT резюме` | Исключить термин |
| Подстановочный знак | `pass*` | Поиск по префиксу |

### Экспорт результатов поиска

После выполнения поиска нажмите кнопки экспорта над результатами:

- **Экспорт CSV** — файл с разделителями-запятыми
- **Экспорт JSON** — структурированный JSON со всеми метаданными
- **Экспорт PDF** — форматированный PDF-отчёт

## Защита бренда

Модуль защиты бренда отслеживает тайпсквоттинг и фишинговые домены, имитирующие ваш бренд.

### Добавление наблюдения за брендом

1. Перейдите в раздел **Защита бренда**
2. Нажмите **Добавить бренд**
3. Заполните:
   - **Название бренда**: отображаемое имя (например, «ООО Акме»)
   - **Официальный URL**: ваш домен (например, `https://acme.ru`)
   - **Ключевые слова**: через запятую
   - **Порог схожести**: минимальный % схожести для генерации алерта (по умолчанию: 70%)
   - **Расписание сканирования**: Ежечасно, Ежедневно или Еженедельно
4. Нажмите **Добавить бренд**

### Просмотр алертов

1. Нажмите **Детали** на карточке бренда
2. В таблице алертов показано:
   - **Найденный домен**: обнаруженный подозрительный домен
   - **Схожесть**: насколько он похож на ваш бренд (круговой индикатор)
   - **Источники**: DNS, WHOIS, SSL, поисковые системы
   - **Статус**: Новый / Проверено / Отклонён
3. Действия:
   - **Проверено**: пометить как изученный
   - **Отклонить**: игнорировать ложное срабатывание
   - **Открыть заново**: вернуть отклонённый алерт

### Ручное сканирование

Нажмите **Глубокое сканирование** в детальном просмотре для немедленного запуска сканирования.

### Экспорт алертов

В детальном просмотре бренда используйте кнопки экспорта (CSV, JSON, PDF).

## Мониторинг (Список наблюдения)

Отслеживайте конкретные индикаторы: IP-адреса, домены, email-адреса, хэши.

### Добавление элемента

1. Перейдите в раздел **Мониторинг**
2. Нажмите **Добавить элемент**
3. Выберите **тип**: IP, Домен, Email, Хэш, Ключевое слово
4. Введите **значение**
5. Выберите **интервал проверки**
6. Нажмите **Сохранить**

---

## Уведомления

Уведомления в реальном времени отображаются в значке колокольчика (вверху справа).

- Нажмите колокольчик для просмотра всех уведомлений
- Нажмите **Отметить все как прочитанные** для сброса счётчика

## Устранение неполадок

**Поиск не возвращает результатов**
- Убедитесь, что файл проиндексирован (раздел «Файлы» — зелёный индикатор)
- Попробуйте упростить запрос

**Сканирование бренда не показывает алертов**
- Попробуйте снизить порог схожести до 60%
- Запустите **Глубокое сканирование** вручную

**Загрузка файла завершается ошибкой**
- Проверьте максимальный размер файла (по умолчанию: 100 МБ)
- Убедитесь, что формат файла поддерживается
